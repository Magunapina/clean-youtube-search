# Clean YouTube Search

A YouTube search app using the YouTube Data API.

The official YouTube search results are often cluttered with unrelated content. Additionally, some sorting options have been removed recently.
By using the YouTube Data API, it delivers clean search results without the noise, and brings back essential sorting features.

## Usage

You must provide your own YouTube Data API v3 key to use this application.
You can obtain one from the [Google Cloud Console](https://console.cloud.google.com).
You need to enter your API key in the sidebar.

## Development

1. Install dependencies:
   ```bash
   bun install
   ```
2. Start the development server:
   ```bash
   bun run dev
   ```
