# Clean YouTube Search

A clean YouTube search app using the YouTube Data API.

## Usage

You must provide your own YouTube Data API v3 key to use this application. You can obtain one from the [Google Cloud Console](https://console.cloud.google.com). You need to enter your API key in the sidebar.

## Development

1. Install dependencies:
   ```bash
   bun install
   ```
2. Start the development server:
   ```bash
   bun run dev
   ```
